# Barbearia Prime

Projeto feito com HTML, CSS e JavaScript para praticar selecao de servicos em uma barbearia.

## O que a pagina faz

- Mostra cards com os servicos da barbearia.
- Permite adicionar servicos ao pedido.
- Atualiza o valor total em tempo real.
- Exibe uma mensagem ao finalizar a selecao.
- Limpa o pedido depois da finalizacao.

## Conceitos usados

- `document.querySelectorAll()`
- `document.getElementById()`
- `addEventListener("click")`
- `parseFloat()`
- Arrow Function
- `forEach()`

## Arquivos

- `index.html`: estrutura da pagina
- `styles.css`: visual da pagina
- `script.js`: logica da soma e finalizacao

## Como abrir

1. Abra `index.html` no navegador.
2. Clique em `Adicionar` nos servicos desejados.
3. Veja o total ser atualizado.
4. Clique em `Finalizar selecao` para concluir e limpar o pedido.
